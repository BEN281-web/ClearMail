ClearMail – AI Email Formatter & Generator

ClearMail is an AI-powered web application that converts informal or unstructured text into well-formatted, professional emails. It helps users quickly draft clear, polite, and structured messages with minimal effort.

🚀 Features
✍️ Convert casual text into professional email format
🎯 Multiple tone options (e.g., Formal)
⚡ Real-time AI-powered rewriting
📋 Copy generated email instantly
🧹 Clear input/output with one click
🧠 Simple, clean UI for fast usage
🛠️ Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
AI Integration: OpenAI API
Tools: Vercel / Render (deployment), Git

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/YOUR_USERNAME/ClearMail.git
cd ClearMail
2. Setup Frontend
cd react
npm install
npm run dev
3. Setup Backend

Open a new terminal:

cd server
npm install
node index.js
4. Environment Variables (IMPORTANT)

Create a .env file inside the server folder:

OPENAI_API_KEY=your_api_key_here
🧠 How It Works
User enters a rough or informal email
Request is sent to backend (Node.js server)
Backend calls OpenAI API
AI returns a refined, professional email
Frontend displays the improved version
